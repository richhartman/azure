Recovery Services Vault

This resource will move with conditions.

Microsoft.Recovery/vaults
Resource Group Movable: Yes
Subscription Movable: Yes:

Disable backups from the Recovery Services Vault before moving.  Disable the Soft Delete setting then delete all Restore Point Collections.  Only one Recovery Services Vault per region may be moved at a time.  Moving Recovery Services Vaults for Azure Backup across Azure Regions is not supported.  For Azure Site Recovery, disable and rebuilt in the new target region. Instructions for disabling Soft Delete can be found here.

Resource Providers

Resource Providers are services.

Example Name: Microsoft.ResourceProvider
Resource Group Movable: NA
Subscription Movable: NA

Every resource type in Azure requires a specific Resource Provider.  Ensure the corresponding Resource Providers are registered in the new Subscription before moving.  For example, a Virtual Machine resource will require the Microsoft.Compute Resource Provider be registered in the target Subscription before it can be moved.  This Resource Provider is registered by default.

This resource will not move.

Microsoft.Compute/restorepointcollections

Restore Point Collections are hidden resources that cannot be moved.  Disable Backups from the Recovery Services Vault.  Disable the Soft Delete setting in the Recovery Services Vault then delete all Restore Point Collections.  Instructions for disabling Backup can be found here.

Instructions for disabling Soft Delete can be found here.
